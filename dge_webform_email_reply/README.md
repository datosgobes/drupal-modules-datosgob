Description
-----------
This module provides a way for users to reply to Contact webform submissions within the
CMS by adding a 'reply' form on webform submissions view. Permission can be set to allow users to reply to Contact webform submissions.
All emails sent are stored in the database and can be viewed
from the submission page through 'replies' tab.


Requirements
------------
Drupal 8.x
Webform 8.x

Installation
------------
2.Login as an administrator. Enable the module in the "Administer" -> "Extends".

3.Enable the required permission:
  View and send email replies to all webforms
    Allows user to view and send emails in reply to any webform.

4.Now when looking at a webform submission you'll see a reply form and
'replies' tab on the page.
